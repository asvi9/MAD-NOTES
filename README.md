---

## 1. Course Information  
- **Course Code:** 303105379  
- **Prerequisite:** Basic knowledge of Java programming  
- **Teaching Mode:** Lecture-based (no lab/tutorial components)  
- **Credits:** 3  
- **Assessment Type:** Theory-based with internal and external examination components  

---

## 2. Course Objectives  
This course is designed to provide students with a structured understanding of mobile application development for the Android platform. The curriculum covers foundational concepts, development tools, architectural patterns,
and practical implementation techniques required to build functional, scalable, and user-centric mobile applications. Emphasis is placed on both native Android development and integration with web services and data storage solutions.

## 5. Learning Outcomes  
Upon successful completion of this course, students will be able to:

1. **Comprehend** mobile application development ecosystems, including architecture, tools, and terminology.  
2. **Design** intuitive and responsive user interfaces using Android UI components and Material Design principles.  
3. **Develop** functional mobile applications with integrated data persistence and web service connectivity.  
4. **Deploy** applications to Android devices and distribute them through public or private marketplaces.
